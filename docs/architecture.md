# 架构设计

## 系统架构

```
┌─────────────┐     ┌─────────────┐     ┌─────────────┐
│   Vue3 前端  │────▶│  FastAPI    │────▶│  ML 模型    │
│  ElementPlus │◀────│   后端服务   │◀────│ SVM / VLM  │
└─────────────┘     └─────────────┘     └─────────────┘
                           │
                           ▼
                    ┌─────────────┐
                    │  数据存储    │
                    │ JSON / CSV  │
                    └─────────────┘
```

## 技术栈

| 层级 | 技术 | 用途 |
|------|------|------|
| 前端 | Vue3 + TypeScript | 用户界面 |
| 前端 | Element Plus | UI 组件库 |
| 后端 | FastAPI | REST API |
| 模型 | scikit-learn | SVM 情感分析 |
| 模型 | PyTorch | VLM 视觉模型 |
| 数据处理 | Jieba | 中文分词 |

## 数据流

1. **导入**: HTML → 解析 → 提取文本/图片
2. **分析**: 文本 → 分词 → TF-IDF → SVM → 情感标签
3. **可视化**: 分析结果 → 统计 → 图表展示
